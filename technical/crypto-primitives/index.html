    Cryptographic Primitives¶

    This document details the cryptographic building blocks used in ZkSnap.

    Homomorphic Encryption¶

    Paillier Cryptosystem¶

    1. Key Generation 

      Generate two large primes p, q where:
- length(p) = length(q)
- gcd(pq, (p-1)(q-1)) = 1

Compute:
- n = pq
- λ = lcm(p-1, q-1)
- g in Z*_n²
- μ = (L(g^λ mod n²))^(-1) mod n
  where L(x) = (x-1)/n

Output:
- Public key: (n, g)
- Private key: (λ, μ)

    2. Encryption 

      Input:
- Message m in Z_n
- Random r in Z*_n
- Public key (n, g)

Compute:
c = g^m * r^n mod n²

Output:
- Ciphertext c

    3. Decryption 

      Input:
- Ciphertext c
- Private key (λ, μ)

Compute:
m = L(c^λ mod n²) * μ mod n

Output:
- Message m

    PLUME Nullifiers¶

    Nullifier Generation¶

    1. Components 

      Input:
- Private key sk
- Message m (vote data)
- Domain separator d

Properties:
- Deterministic output
- Unique per (key, message)
- Non-invertible

    2. Generation Process 

      Steps:
1. h = Hash(m || d)
2. s = PLUME_sign(sk, h)
3. N = Hash(s)

Output:
- Nullifier N
